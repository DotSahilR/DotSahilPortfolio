AlzAware

Alzheimer’s detection using brain scan analysis

A multi-model system combining CNN and LSTM to analyze MRI and PET scans for early brain disease detection. The model captures both spatial and temporal features, improving diagnostic accuracy. A user-friendly web interface allows users to upload scans and receive predictions in real time. This project supports researchers and healthcare professionals with accessible, AI-powered diagnostic assistance. The frontend was built using HTML, CSS, JavaScript, and Tailwind CSS, while the backend uses Flask in Python for model integration and API handling.
